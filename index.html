About Me

I'm a passionate software developer that is always learning something new.

Senior Engineer with 10 years of professional experience in designing high quality object oriented software employed in real time operating systems for real time applications. Skilled C++ programmer with vast experience in embedded system design and device driver development. Equally skilled in Python for developing multi-process pc application that serves as access point to control a number of slave boards. Experienced in web development both front-end (HTML, CSS, JS, jQuery) and server side (Python, Flask, Django, PHP, Node.js). Hobby android app development using Kotlin. Playing around with front-end frameworks Angular, VueJS, and aurelia. Experienced in leading a team to deliver high quality software.
